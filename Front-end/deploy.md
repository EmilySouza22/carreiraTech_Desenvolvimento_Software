# Publicando o site com o GitHub

## Passos para o Deploy

- Dentro do seu repositório no GitHub vá na aba chamada `Settings`
- Dentro de Settings vá na aba `Pages`

Agora escolha o seguinte:
- Para a seção `source` pode deixar o padrão que vem **Deploy from a branch**
- Para `Branch` coloque a branch principal `Main`
- Clique em salvar

## Como acessar o site?

- Em `Custom domain` irá aparecer a URL do site
- Exemplo de URL: `nickname.github.io/nome_repositorio/`

- Pronto agora o site pode ser acessado por qualquer pessoa!
